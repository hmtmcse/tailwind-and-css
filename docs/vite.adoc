
== Create react project
```bash
pnpm create vite
```

== Add Tailwind CSS
```bash
pnpm add tailwindcss @tailwindcss/vite
```

=== Add the `@tailwindcss/vite` plugin
```js
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    tailwindcss(),
  ],
})
```

=== Import css in any css
```
file: src/App.css
@import "tailwindcss";
```


== Add Shadcn
```bash
pnpm add class-variance-authority clsx tailwind-merge
```

* *class-variance-authority :* Manages component variants (e.g., size, color, state) in a clean, scalable way.
* *clsx :* Combines class names conditionally
* *tailwind-merge :* Resolves Tailwind CSS class conflicts (e.g., px-2 px-4 → px-4).
